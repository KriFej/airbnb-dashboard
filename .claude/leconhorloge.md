# Leçons Horloge — Estimations de temps

> Lire ce fichier AVANT chaque estimation. Ne pas répéter les mêmes erreurs.

## Format d'un log
- **Tâche** : description
- **Estimé** : X min
- **Réel** : Y min
- **Erreur** : +/- Z min
- **Pourquoi** : explication
